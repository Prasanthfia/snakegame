# 🐍 Snake Game

A classic Snake game with a neon green dark theme.

## Features
- Smooth snake movement with arrow key controls
- Levels that increase speed as you score
- High score tracking in the session
- Pause, reset, and restart controls
- Glowing neon grid design

## Controls
- Arrow Keys — Move the snake
- Space — Start / Pause
- Restart button — Reset the game

## Built With
- HTML5 Canvas
- CSS3
- Vanilla JavaScript

## How to Run
Clone the repo and open `index.html` in your browser.
